# Nexus Vision
**Tactical Home Network Intelligence Platform**

---

## الميزات

### الرادار
- رادار دوار بخط مسح (Sweep) حقيقي بزاوية 360°
- كل هاتف يظهر كنقطة (Blip) على الرادار
- تأثير Glow فوسفوري ثلاثي الطبقات حول كل نقطة
- Fade تدريجي: النقطة تضيء عند مرور الخط وتتلاشى ببطء
- المتطفلون يظهرون بلون أحمر نابض
- قوة الإشارة (RSSI) تحدد موقع النقطة: الأقرب = المركز

---

### اكتشاف الأجهزة
- قراءة جدول ARP الحقيقي (لا بيانات افتراضية)
- يعرض فقط هواتف iPhone و Android
- لكل هاتف: الاسم، الشركة المصنّعة، نظام التشغيل، الـ IP، الـ MAC، قوة الإشارة
- فحص المنافذ المفتوحة (Port Scan) لكل هاتف تلقائياً عند اكتشافه
- تحديث OUI (200+ مدخل) لتحديد الشركة المصنّعة بدقة
- على Android: قراءة RSSI الحقيقي من WiFiManager

---

### مراقبة الحركة الشبكية
- قراءة جدول TCP النشط (/proc/net/tcp على Android/Linux)
- تحليل DNS لمعرفة المواقع والخدمات المستخدمة
- تصنيف الاتصالات: WhatsApp / Instagram / YouTube / TikTok / iCloud...
- تحديد اتجاه الاتصال: إرسال (OUT) أو استقبال (IN)
- سجل تفصيلي لكل هاتف: الوقت، الخدمة، الـ hostname، نوع النشاط

---

### مركز القيادة (Command Center)
- بطاقة شبكة علوية: IP الراوتر، IP جهازك، عدد الأجهزة المتصلة
- رابط مباشر للوحة إدارة الراوتر
- لكل هاتف:
  - **KICK** — طرد الجهاز من الشبكة (يتطلب Admin/Root)
  - **UNBLOCK** — إعادة الجهاز للشبكة
  - **Speed Limit** — تحديد السرعة: 256K / 512K / 1MB / 5MB / FREE
  - **PING** — اختبار الاتصال الفوري
  - **Copy IP / Copy MAC** — نسخ البيانات للحافظة
- قائمة الأجهزة المحجوبة مع زر رفع الحجب

---

### مراقبة الحالة (PingMonitor)
- يُعيد Ping لكل جهاز معروف كل 8 ثوانٍ تلقائياً
- يعرض حالة ONLINE / OFFLINE بشكل لحظي
- يستخدم TCP Connect كبديل إذا فشل ICMP ping

---

### نظام التنبيهات
- صوت تنبيه مُولّد برمجياً (بدون ملفات خارجية)
- اهتزاز عند ظهور جهاز جديد (Android)
- إشعار (Notification) على Android والـ Desktop
- معدل تحديد: لا يتكرر التنبيه لنفس الجهاز أكثر من مرة كل 10 ثوانٍ

---

### كشف المتطفلين (Intruder Detection)
- قائمة بيضاء (Whitelist) للأجهزة الموثوقة
- تفعيل/تعطيل وضع الكشف من الإعدادات
- أي هاتف خارج القائمة البيضاء يُصنَّف تلقائياً "INTRUDER"
- يظهر بلون أحمر في الرادار والشاشات
- تنبيه فوري عند اكتشاف متطفل
- زر TRUST / UNTRUST لكل جهاز

---

### اختبار سرعة الإنترنت
- قياس Ping (ms)
- قياس Download (Mbps)
- قياس Upload (Mbps)
- بدون مكتبات خارجية — يستخدم urllib وsocket فقط

---

### الإعدادات
- سرعة الفحص: 5s / 10s / 15s / 30s / 60s
- تفعيل/تعطيل تحليل DNS
- تنبيه جهاز جديد: ON/OFF
- تنبيه نشاط مجهول: ON/OFF
- حفظ سجل الأحداث: ON/OFF
- حفظ سجل الاتصالات: ON/OFF
- مسح قاعدة البيانات بالكامل
- إدارة القائمة البيضاء

---

### قاعدة البيانات
- ملف JSON محلي (لا اتصال خارجي)
- حفظ متأخر (Debounced) لتقليل الكتابة على القرص
- يخزن: الأجهزة، السجلات، الاتصالات، الإعدادات، القائمة البيضاء

## الأذونات المطلوبة (Android)
| الإذن | الاستخدام |
|-------|-----------|
| ACCESS_WIFI_STATE | قراءة معلومات الشبكة |
| CHANGE_WIFI_STATE | تفعيل مسح WiFi |
| ACCESS_FINE_LOCATION | مطلوب لمسح WiFi في Android 9+ |
| INTERNET | اختبار السرعة |
| VIBRATE | تنبيهات الاهتزاز |

**Root / Admin مطلوب لـ:** KICK، Speed Limit
